# データセット
## DeepLabCut
DeepLabCutで用いるデータセットは以下のディレクトリに格納されている．<br/>
`minelab-dataset\20250000_HoneyBeeColony_永井\DeepLabCut\`<br/>
DeepLabCutでのアノテーションデータは次の通り定義する．
* 頭部（Head）
* 腹部（Body）
* 針部（Stinger）

また，追加でBodyとStingerの間にAbdomenを追加し，4点を用いることにしたデータセットがある．
これは以下のディレクトリに格納されている．<br/>
`minelab-dataset\20250000_HoneyBeeColony_永井\DeepLabCut\4Kpt(Head,Abdomen,Body,Stinger)`
